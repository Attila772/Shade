# Shade
 Isometric stealth game in [Godot engine (3.3)](https://godotengine.org/download), Mono version.

 Isometric stealth game , where the player has to steal an artifact on each stage 
 without getting caught.
 Player needs to think and plan their gameplay style to trick the guards, or sneak past them using the environment and/or
 other gadgets.
 
## Funkciók
 - Főmenü
 - Főhadiszállás pálya
   - Hordozható eszköz választása
   - Teljesítendő pálya választása
 - Használható eszközök (maximálisan 1 eszköz limit)
 - Őrök
   - Járőrözés (megadott útvonalon)
   - Üldözés (törekvés a bekerítésre)
   - Lehetséges búvóhelyek kutatása
   - Riasztottsági szint (üldözés után)
 - Kamerák
   - Pásztázás
   - Őrök azonnali riasztása
 - Zárt ajtók és ezeket nyitó kulcs
